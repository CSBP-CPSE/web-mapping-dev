LODE Viewer

A Web Viewer for the Linkable Open Data Environment (LODE), including the Open Database of Buildings (ODB). The viewer currently provides maps for different indicators. The links are provided below :

Population to building ratio Building provenance (Microsoft or ODB) Proximity to public transit Proximity to libraries

The viewer is developed and maintained by the Data Exploration and Integration Laboratory at Statistics Canada. The code is available on our GitHub page.

Visionneuse de l'ECDO

Une visionneuse pour les données de l'ECDO, incluant la base de données ouvertes de bâtiments (BDOB). La visionneuse offre présentement des cartes pour différents indicateurs. Les liens sont fournis ci-bas:

Proportion de la population au nombre de bâtiments Origine des bâtiments (Microsoft ou BDOB) Proximité aux transports en commun Proximité aux bibliothèques

La visionneuse est développée et entretenue par le Laboratoire d'exploration et intégration de données à Statistique Canada. Le code est disponible sur notre page GitHub.
